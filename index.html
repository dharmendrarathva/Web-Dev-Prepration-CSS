Box Model

Margin Border Padding Content
Four layers of an element's box.
1. Margin
Space outside the element, separating it from others.

   .box{
          width: 200px;
          height: 100px;
       }
2. Border
Line around padding + content.

   .box{
         border: 2px solid black;
       }
3. Padding
Space inside the element, between content and border.

   .box{
        padding: 20px;
       }
4. Content
Actual text, image, or element content.

   .box{
         width: 200px;
         height: 100px;
       }

Box-Sizing

1. content-box (Default)
width applies only to content
Padding and border increase total size
Total width = width + padding + border 
    
      .box {
              width: 200px;
              padding: 20px;
              border: 5px solid;
              box-sizing: content-box;
            }
200 + 40 + 10 = 250px
2. border-box (Recommended)
width includes content + padding +border
Total size remains constant
Total width = width 
    
      .box {
              width: 200px;
              padding: 20px;
              border: 5px solid;
              box-sizing: border-box;
            }
200px (fixed)
Best Practice :
Use border-box for easier sizing and layout control.
          
            * {
                box-sizing: border-box;
              }
