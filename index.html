Foundation (CSS Basics)

CSS Syntax
Defines how styles are written.
Selectors, Properties, Values
Selectors target HTML elements; properties define what to style; values specify the style.
CSS Comments
Used to add notes in CSS code.
Inline, Internal, External CSS
Three methods to apply CSS to HTML.
Cascade & Inheritance
Cascade : When multiple rules apply, CSS decides which one wins.
Inheritance : Child elements inherit some styles from parents.
Specificity Rules
Decides which selector is more powerful Priority.
Selector Type Specificity Score
Universal * 0
Element p , div 1
.class 10
Attribute [type="text"] 10
Pseudo-class :hover 10
#id 100
Inline Css 1000 
        Best Practice (Real Projects) :
      
        ✔ Prefer class selectors
        ✘ Avoid using ID selectors
        ✘ Minimize inline styles
        ✘ Avoid !important
!important Usage
Forces a rule to override all others. Avoid Large projects (breaks maintainability)

Selectors

Universal (*) Selector
Selects all elements on the page.
Removing or neutralizing the browser’s default styles
Element Selector
Selects all elements of a given tag.
Class (.class) and ID (#id) Selectors
Class: Selects elements with a specific class attribute.
ID: Selects a unique element with a specific id attribute.
Grouping Selectors
Combines multiple selectors to apply the same styles.
Descendant
Targets elements inside another element (any depth).
Child (>)
Targets direct children of an element.
Adjacent (+)
Selects the immediately next sibling.
